# AI Resume Screening System

## Overview
An AI-based Resume Screening System built using Python, Flask, and MongoDB. It analyzes resumes, compares them with job descriptions, calculates a matching score, identifies missing skills, provides AI suggestions, stores candidate data, and generates PDF reports.

## Features
- Resume Upload (PDF)
- Resume Parsing
- Skill Matching
- Resume Score Calculation
- AI Suggestions
- MongoDB Database
- Recruiter Dashboard
- Candidate Search
- Dashboard Statistics
- PDF Report Generation

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- MongoDB
- ReportLab

## How to Run
### 1. Install requirements:
```bash
pip install -r requirements.txt
```

### 2. Run the application
```bash
python app.py
```

### 3. Open in browser
```text
http://127.0.0.1:5000
```
